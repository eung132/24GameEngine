### VCS란
* "Version Control System"의 약어로, 소프트웨어 개발에서 
    코드 및 파일의 변경 내역을 추적하고 관리하는 도구입니다.
    
### DVCS와 VCS의 차이점
* DVCS는 분산 구조를 가지고 있으며, 
    각 개발자는 로컬 저장소를 가지고 작업을 하며 변경 사항을 
    원격 저장소에 반영할 수 있습니다.
    Git과 Mercurial이 이에 해당합니다. VCS는 중앙 집중식 구조를 가지며, 
    모든 파일의 이력과 변경 사항이 중앙 서버에 저장됩니다. 
    SVN과 CVS가 이에 해당합니다.
